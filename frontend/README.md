Getting Started with Event Recommendation System - Frontend (React.js)

This project was bootstrapped with Create React App.

--------------------------------------------------------
Setting Up the Node Environment:

To run this project, ensure you have Node.js installed on your machine. You can download it from:
https://nodejs.org/

After installing Node.js, follow these steps to set up the project:

1. Open a terminal or command prompt.
2. Navigate to the project directory:
   ```bash
   cd C:\Users\EventRecommendationSystem\frontend
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

--------------------------------------------------------
Available Scripts:

1. npm start:
   - Runs the app in development mode.
   - Open http://localhost:3000 to view it in the browser.

2. npm test:
   - Launches the test runner in interactive watch mode.

3. npm run build:
   - Builds the app for production in the 'build' folder.
   - Minified and optimized for best performance.

4. npm run eject:
   - Use this for full control over the configuration.
   - **Warning:** Irreversible action.
